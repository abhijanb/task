
# Home Take Task
## backend: flow

cd server
pnpm install or npm install
copy .env copy to .env and fill the values
pnpm dev or npm run dev
visit /seed to seed the database for properties


## frontend:  flow
cd client
pnpm install or npm install
copy .env.copy to .env and fill the values
e.g.
VITE_BASE_API_URL=http://localhost:3000/api

pnpm dev or npm run dev
visit frontend at http://localhost:5173/register to register a user and then login to see the dashboard
